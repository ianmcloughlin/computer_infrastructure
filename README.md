# Computer Infrastructure

This repository contains details for the assessment period July-August 2025.

[![Adapted from Art by Matt Leadbeater](https://raw.githubusercontent.com/ianmcloughlin/images/master/banner_computer_infrastructure.png)](https://www.instagram.com/mattleadbeater/)

## Purpose

Learn how to:

1. Use, configure, and script in a command line interface environment.

2. Manipulate and move data and code using the command line.

3. Compare commonly available software infrastructures and architectures.

4. Select appropriate infrastructure for a given computational task.
